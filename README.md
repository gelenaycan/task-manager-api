# Task Manager API (Ktor + Kotlin)

A simple and clean REST API built using **Ktor**, **Kotlin**, and **in-memory storage**.  
This project demonstrates the fundamentals of building backend services using modern Kotlin tools.

Perfect for:
- learning server-side Kotlin  
- practicing REST API design  
- preparing for junior backend interviews  



##  Tech Stack

- **Kotlin**
- **Ktor 3.x**
- **Netty Engine**
- **Kotlinx Serialization (JSON)**
- **Gradle Kotlin DSL**
- **In-memory database (MutableList)**
